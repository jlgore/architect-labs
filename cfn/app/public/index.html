API Documentation

CloudMart provides a RESTful API for product management. Here are some example commands:

Get all products
curl -X GET http:///api/products
Get a specific product
curl -X GET http:///api/products/1
Create a new product
curl -X POST http:///api/products \
-H "Content-Type: application/json" \
-d '{"name":"New Product","price":99.99,"description":"Product description","image":"product-image.jpg"}'
Update a product
curl -X PUT http:///api/products/1 \
-H "Content-Type: application/json" \
-d '{"name":"Updated Name","price":129.99}'
Delete a product
curl -X DELETE http:///api/products/1

About CloudMart

Our Mission

At CloudMart, we're passionate about providing high-quality cloud products at affordable prices. Our mission is to make cloud technology accessible to everyone.

This demo application shows how to integrate a frontend with a backend API running on AWS infrastructure deployed via CloudFormation.

AWS Architecture

This application is powered by:

  • EC2 instances in an Auto Scaling Group
  • Application Load Balancer for traffic distribution
  • RDS MySQL database for product storage
  • VPC with public and private subnets
